AOC 2022 in Scala3

# Commands

## To download description and input
`./aoc.sh <DAY>`

## To test
`./aoc.sh test <DAY>`

## To submit
`./aoc.sh submit <DAY> <PART> <ANSWER>`

## To download part 2
`./aoc.sh part2`

## To check code formatting
`./aoc lint`
